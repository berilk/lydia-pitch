# LYDIA Pitch Website

This is a static presentation website for LYDIA, designed to be deployed without a build step.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

Upload the full `lydia-pitch` folder to GitHub Pages, Vercel, Netlify, or any static host. The entry file is `index.html`.

## Presenting

- Use the right arrow, down arrow, Page Down, or Space to advance.
- Use the left arrow, up arrow, or Page Up to go back.
- Use Home and End to jump to the beginning or end.
