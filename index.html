Technical Details & MVP Scope

Merzifon Municipality context - Amasya Province - July 03, 2026.

MVP prototype: simulated LeakDB data, realistic topology, not yet a production municipal deployment.
Model XGBoost

Incremental training on LeakDB Hanoi network simulations.

Training scale ~12.3M

Training steps from 700 scenarios and 161 features.

Test set 2.63M

Steps from 150 held-out scenarios never seen in training.

Infrastructure FastAPI + Streamlit

Cloud-hosted prediction API and deployed monitoring interface.
