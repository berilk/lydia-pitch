MVP - Implemented

  • Real-time anomaly detection across 32 measurement points.
  • Live deployed backend and frontend, not a local demo.
  • XGBoost model trained on 700 LeakDB scenarios, about 12.3M steps.
  • Held-out test set of 150 scenarios, 2,628,000 steps.
  • EPANET Hanoi topology with 32 nodes and 34 pipes.

Next - In Development

  • Real municipal SCADA and sensor integration.
  • Signed pilot partnerships with municipalities.
  • Dedicated field team mobile app and dispatch system.
  • Pipe-segment level leak localization.
